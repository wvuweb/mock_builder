Table of Contents

This is a style guide for slate themes. The purpose is to ensure that styles are created for various page elements. It is not a guarantee that the styles displayed here will appear the same way in templates.

The guide is broken down into the following sections:

  • Paragraphs (as well as inline style elements)
  • Links
  • Headings
  • Lists
  • Images
  • Tables
  • Forms

Paragraphs (#)

This is an example paragraph. Within this paragraph, we have a number of inline style elements. For example, we have strong text and bolded text. We have emphasized text and italicized text as well. We also have spanned text, which can be useful for highlighting text.

Don’t forget about citations, such as Cat’s Cradle by Vonnegut. Or acronyms such as CSS.

Just for completeness, there’s also inserted text, deleted text, superscript text, subscript text, and code.

Links (#)

There are four states for links that can be styled: link, visited, hover, and active. You can also combine these states in order to style hovering over a visited link. The four states have different levels of precedence. A helpful way to remember the precedence is the phrase LoVeHAte, where the capital letters are for Link, Visited, Hover, Active (Link having the least precedence).

  • link – unvisited link
  • visited – click this link to test visited style
  • hover – mouse-over to test hover style
  • active – hold mouse button down to see the active style
